# Roadmap

## Phase 1

- Establish repo, tooling, docs, tests, and initial graph-first shell.
- Define canonical domain types and fixture-backed pipeline contracts.
- Prove one end-to-end transformation path from raw fixture to graph view model.

## Phase 2

- Replace the graph preview with a real Sigma.js scene.
- Add camera controls, zoom behaviors, node focus states, and richer detail panels.
- Introduce filter state for eras, divisions, and events.

## Phase 3

- Expand ADCC ingestion toward a fuller dataset with provenance notes.
- Add deeper network metrics such as centrality and community detection.
- Surface rivalry timelines and bridge-athlete analysis.

## Phase 4

- Add inferred matchup candidates and “who never fought whom” exploration.
- Introduce multi-source support for rankings and athlete metadata.
- Improve storytelling with timelines, narratives, and cinematic transitions.
